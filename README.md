# MAB-PE-DCOP

We used the pyDCOP framework to construct a new algorithm, heist, to solve DCOP (Decentralized Constraint Optimization Problems)

# Running Application

To run the program do the following:
1) cd pyDCOP
2) python3 xmlsolve.py xml_solve --algo heist 'file_path'
3) press ctrl + c when done

# Installing

To install simply clone in the desired folder using the git clone command

# Dependencies

It is required to have python 3.0+. To install python 3, please follow the direction on the python website: https://www.python.org
